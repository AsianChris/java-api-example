# Java API Example using Jersey

This is an example of a Java API written using Jersey. This example was built as a Java servlet, and tested with Tomcat 7.

## Requirements

* Java 7
* Apache Maven 3

## Tested using

* Ubuntu 14.04 LTS
* Apache Tomcat 7
* Java 7 OpenJDK
* Maven 3.0.5
